Getting Started with CUAHSI QA Workstations

  1. Go to the Code Server IDE at
  2. Open Folder, and choose /home/coder/
  3. Open a terminal, from the Hamburger Icon > Terminal > New Terminal
  4. git clone https://github.com/CUAHSI/QA-AutomationEngine
  5. cd QA-AutomationEngine
  6. Install dependencies with pip3 install -r requirements.txt
  7. Run ./hydrotest hydroshare HydroshareTestSuite --grid selenium --browser chrome
  8. Use and to watch test execution and the test node status, respectively
